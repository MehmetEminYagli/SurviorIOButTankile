📋 IAttackStrategy.cs

📋 Interface 🎯 Strategy ⚔️ Combat 

public interface IAttackStrategy
{
    void Attack(Transform target, float accuracy);
}

📝 Arayüz Açıklaması

  • 🎯 Temel Amaç: Farklı saldırı stratejileri için ortak bir sözleşme tanımlar
  • 🔧 Özellikler:
    • Strategy Pattern implementasyonu
    • Saldırı davranışı standardizasyonu
    • Modüler ve genişletilebilir yapı
    • Bağımlılık tersine çevirme prensibi

🔄 Attack Metodu


void Attack(Transform target, float accuracy);
  • Parametreler:
    • target: Hedef Transform
    • accuracy: İsabet oranı (0-1)
  • Kullanım:
    • Hedef nesneye saldırı gerçekleştirme
    • İsabet oranına göre saldırı etkisini ayarlama
    • Farklı saldırı tipleri için temel şablon

💡 Uygulama Notları

  • Kullanım Alanları:
    • Yakın mesafe saldırıları (MeleeAttackStrategy)
    • Uzak mesafe saldırıları (RangedAttackStrategy)
    • Özel saldırı tipleri için genişletilebilir
  • Tasarım Prensipleri:
    • Strategy Pattern
    • Dependency Inversion
    • Single Responsibility

🔄 Mevcut Implementasyonlar

Sınıf Açıklama Özel Özellikler
MeleeAttackStrategy Yakın mesafe saldırıları
  • Mesafe kontrolü
  • Doğrudan hasar
RangedAttackStrategy Uzak mesafe saldırıları
  • Mermi sistemi
  • Sapma hesaplama

📝 Kullanım Örneği


public class Enemy : MonoBehaviour
{
    private IAttackStrategy attackStrategy;

    private void Awake()
    {
        // Strateji ataması
        attackStrategy = GetComponent();
    }

    public void PerformAttack(Transform target)
    {
        // Strateji kullanımı
        attackStrategy.Attack(target, accuracy);
    }

    public void ChangeStrategy(IAttackStrategy newStrategy)
    {
        // Runtime'da strateji değiştirme
        attackStrategy = newStrategy;
    }
}
