⚔️ Melee 🤖 AI 🎯 NavMesh 👾 Enemy 

public class MeleeEnemy : BaseEnemy
{
    protected override void Awake()
    {
        base.Awake();
        attackStrategy = GetComponent();
        if (attackStrategy == null)
        {
            attackStrategy = gameObject.AddComponent();
        }

        // Set default values for melee enemy
        stoppingDistance = 2f;
        attackRange = 2.5f;
        accuracy = 0.9f;
        moveSpeed = 7f;
        agent.stoppingDistance = stoppingDistance;
        agent.speed = moveSpeed;
    }
}

📝 Sınıf Açıklaması

  • 🎯 Temel Amaç: Yakın dövüş düşman tipini temsil eder ve BaseEnemy'den türer
  • 🔧 Özellikler:
    • Yakın mesafe saldırı stratejisi
    • Hızlı hareket kabiliyeti
    • Kısa durma mesafesi
    • Yüksek isabet oranı

⚙️ Varsayılan Değerler

  • Hareket Ayarları:
    • stoppingDistance: Durma mesafesi (2)
    • attackRange: Saldırı menzili (2.5)
    • accuracy: İsabet oranı (0.9)
    • moveSpeed: Hareket hızı (7)
