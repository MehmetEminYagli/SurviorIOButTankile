🏹 RangedEnemy.cs

🏹 Ranged 🤖 AI 🎯 NavMesh 👾 Enemy 

public class RangedEnemy : BaseEnemy
{
    protected override void Awake()
    {
        base.Awake();
        attackStrategy = GetComponent();
        if (attackStrategy == null)
        {
            attackStrategy = gameObject.AddComponent();
        }

        // Set default values for ranged enemy
        stoppingDistance = 10f;
        attackRange = 15f;
        accuracy = 0.7f;
        agent.stoppingDistance = stoppingDistance;
    }
}

📝 Sınıf Açıklaması

  • 🎯 Temel Amaç: Uzak mesafe düşman tipini temsil eder ve BaseEnemy'den türer

⚙️ Varsayılan Değerler

  • Hareket Ayarları:
    • stoppingDistance: Durma mesafesi (10)
    • attackRange: Saldırı menzili (15)
    • accuracy: İsabet oranı (0.7)
